#Anagram

###Specs

This program will...

1. Break a word into its individual letters.
  Input: "tab"
  Output: {"t", "a", "b"}
2. Arrange the array of letters in alphabetical order.
  Input: {'t', 'a', 'b'}
  Output: {'a', 'b', 't'}
3. Determine if two arrays of letters are equivalent.
  Input: {'a', 'b', 't'}, {'a', 'b', 't'}
  Output: true
